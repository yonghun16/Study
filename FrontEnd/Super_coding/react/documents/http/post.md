## POST request


### POST 요청
- 기본적으로 서버에 데이터를 보낼 때 사용하는 HTTP 메소드
- 일반적으로 파일 업로드, Form 형태의 제출에서 많이 사용한다.

### Header
- Content-Type: application/x-www-form-urlencoded
  - body: key=value&key=value
- Content-Type: application/json
  - body: key: value

### Google Firebase
- 더미 데이터베이스를 만들어서 POST 요청을 보낼 수 있음.
